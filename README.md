# js-variables

Esercizi base su variabili, operatori e stringhe in JavaScript.

## Obiettivo

Completare tutte le task richieste nei file `script.js`, verificando il risultato tramite `console.log` e DevTools del browser.

## Contenuti

Gli esercizi riguardano:

- dichiarazione di variabili
- tipi primitivi
- operatori matematici
- concatenazione di stringhe
- proprietà delle stringhe
- metodi principali delle stringhe
